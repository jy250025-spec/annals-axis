# 纪年轴 (annals-axis)

> 基于时间轴的笔记记录桌面应用 — 以编年体的方式整理你的知识。

## 简介

纪年轴是一个面向历史阅读和个人笔记的场景而设计的桌面工具。它以**时间轴（Timeline）**为核心组织方式，让你可以按年、月、日记录和查阅笔记，适合阅读史书、整理编年体资料或个人时间线记录。

## 功能

- **书卷管理** — 创建多个笔记主题（书卷），每个书卷独立管理
- **时间轴展示** — 所有条目按时间顺序在时间轴中展示
- **智能日期解析** — 支持中文日期格式，如 `1933年10月10日`、`3月上旬`、`5月20日`
- **自动排序** — 条目按年月日自动排序
- **暗黑模式** — 支持明暗主题切换
- **侧边栏折叠** — 可收起侧边栏获得更大阅读空间
- **本地存储** — 数据持久化在浏览器本地存储中

## 快速开始

```bash
# 安装依赖
npm install

# 运行开发模式
npm start

# 打包为 Windows 便携版 exe
npm run build
# 输出在 release/ 目录
```

## 使用说明

1. 点击左侧 **+** 按钮创建新的书卷（如"三国志"、"全球通史"）
2. 选中书卷后，在底部的输入框中记录笔记
3. 每条记录必须包含日期。首次输入需要包含年份，如 `1933年10月10日，内容...`
4. 后续输入可省略年份，自动沿用上一条的年份，如 `3月1日，内容...`
5. 记录会自动按时间排序，在时间轴中展示
